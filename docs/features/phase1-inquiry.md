# Phase 1: 問い合わせ・見積自動化

## 概要

Webフォームからの問い合わせ受付 → 見積金額の自動計算 → 見積書PDF生成 → メール自動送付を一気通貫で処理する。

## 問い合わせフォーム（顧客向け）

| 入力項目 | 内容 |
|---------|------|
| 施設選択 | 施設ごとに料金が異なる |
| 利用希望日 | カレンダー選択（平日/休日/休前日を自動判定） |
| 人数 | 人数入力 |
| 会議室利用 | あり / なし |
| 宿泊 | あり / なし |
| 朝食 | あり / なし（宿泊ありの場合のみ表示） |
| 昼食 | あり / なし |
| 夕食 | あり / なし |
| 会社名 | テキスト入力 |
| 氏名 | テキスト入力 |
| メールアドレス | メールアドレス入力 |

## 見積金額の自動計算ロジック

1. 利用希望日を日程マスタと照合し「平日/休日/休前日」を判定
2. 料金マスタ（施設 × 日程区分 × 項目）から単価を取得
3. 単価 × 人数で各項目の金額を算出（税込）
4. 選択された項目のみ見積書に表示

## 見積書PDF自動生成・メール送付

- 入力内容をもとに見積書PDFを自動生成
- SendGrid経由で顧客のメールアドレスへ自動送付
- 施設ごとに差出人ドメイン・署名を切り替え対応

## マスタ管理機能（管理画面）

| 機能名 | 概要 |
|--------|------|
| 料金マスタ管理 | 施設×日程区分×項目の単価を管理画面から設定（税込） |
| 日程マスタ管理 | 日付ごとに「平日/休日/休前日」を設定・管理 |
| 施設マスタ管理 | 施設名・差出人ドメイン・メール署名を設定 |
| メールテンプレート管理 | 見積送付メールの文面を施設ごとに設定 |
